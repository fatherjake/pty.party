# CLAUDE.md

The working agreement for agents on this project lives in [AGENTS.md](AGENTS.md).
**Read it before doing anything and follow it.**

Most importantly: this project keeps a running checklist of work on a connected
**Log** card on the pty.party canvas. Pushing the tasks you're about to do onto
the Log and ticking them off as they finish is the first and last step of any
task — see [AGENTS.md](AGENTS.md) for the full loop and the fallback when the
MCP tools are unavailable.
